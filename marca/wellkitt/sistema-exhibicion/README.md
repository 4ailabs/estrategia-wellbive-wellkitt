# Sistema de exhibición Wellkitt

Carpeta maestra para el diseño, producción e implementación del punto de venta
físico Wellkitt.

## Estructura

### `anaquel/`

Sistema permanente de navegación por categorías:

- Vista frontal proporcional.
- Diseño vectorial editable.
- Medidas y criterios de producción.

### `mesa-semanal/`

Zona temática que cambia cada semana:

- Mockups visuales.
- Character sheet.
- Guía para Cricut.
- Archivos SVG editables.
- Medidas y reglas de composición.

### `espacio/`

Propuestas para organizar el local completo:

- Distribución del mobiliario.
- Circulación.
- Renders generales.

## Carpetas previstas

Cuando comience la producción se añadirán:

- `cricut/`: archivos individuales listos para cortar.
- `impresion/`: artes finales con sangrado.
- `plantillas-semanales/`: diseños editables por categoría.
- `fotografias/`: registro de la implementación real.
- `archivo/`: versiones descartadas o sustituidas.

## Convención de nombres

```text
wellkitt-[zona]-[pieza]-[categoria]-v01.ext
```

Ejemplo:

```text
wellkitt-mesa-cartel-calma-v01.svg
```
