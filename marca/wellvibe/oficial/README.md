# Wellvibe Logo Oficial de Trabajo

Direccion elegida: **sans clinica**.

## Criterio

Logo solo tipografico, sin subtitulo y sin simbolo.

Debe sentirse:

- premium
- clinico
- editorial
- sobrio
- limpio
- confiable

## Version principal

- `wellvibe-logo-principal.svg`

## Versiones

- `wellvibe-logo-principal.svg`: fondo marfil, texto grafito verde.
- `wellvibe-logo-positivo.svg`: transparente, texto grafito verde.
- `wellvibe-logo-negativo.svg`: transparente, texto marfil.
- `wellvibe-logo-youtube.svg`: version centrada para avatar/banner o uso en video.

## Paleta

- Grafito verde: `#111C1A`
- Marfil clinico: `#F6F1E8`
- Salvia gris: `#6C756F`

## Uso

Usar siempre el logo sin subtitulo.

No agregar:

- iconos
- ondas
- circulos
- hojas
- cruces
- descriptor debajo

Para miniaturas de YouTube, usar el logo pequeño en una esquina o como firma discreta.
