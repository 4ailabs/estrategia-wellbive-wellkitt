# Wellvibe Premium 01

Direccion de marca: clinica premium, editorial y sobria.

## Concepto

Wellvibe se presenta como una marca de salud integrativa con autoridad educativa.

La identidad no explica literalmente "vibracion" con ondas o iconos evidentes. La sensacion de ritmo aparece de forma sutil en:

- la `v` central del nombre
- el monograma `wv`
- una linea/acento minimo para aplicaciones

## Personalidad visual

- profesional
- serena
- precisa
- humana
- clinica sin ser fria
- moderna sin parecer startup generica

## Tipografia sugerida

Para implementar en web:

- `Satoshi`
- `Aptos`
- `Inter`
- `Manrope`

Para version final de diseño, conviene explorar una tipografia premium como:

- Neue Haas Grotesk
- Suisse Int'l
- Canela / Editorial New para acentos editoriales

## Archivos

- `wellvibe-wordmark.svg`: logo principal.
- `wellvibe-monograma.svg`: avatar / favicon / sello.
- `wellvibe-lockup.svg`: composicion con descriptor.

## Uso recomendado

Usar el wordmark como marca principal. El monograma solo para avatar, favicon, marca de agua o sello pequeño en miniaturas.
