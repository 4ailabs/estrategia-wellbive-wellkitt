## Decision de marca

Wellvibe usara un wordmark tipografico:

```text
wellvibe
```

Sin simbolo.  
Sin subtitulo.  
Sin ondas.  
Sin hojas.  
Sin cruz.  
Sin elementos wellness genericos.

## Uso recomendado

### YouTube

- Usar `oficial/wellvibe-logo-miniatura-firma.svg` como firma pequeña en miniaturas.
- Usar `oficial/wellvibe-logo-youtube.svg` para pantallas de inicio, cierre o banner.

### Documentos y materiales

- Usar `oficial/wellvibe-logo-principal.svg` sobre fondo claro.
- Mantener espacio limpio alrededor del logo.

### Fondos oscuros

- Usar `oficial/wellvibe-logo-negativo.svg`.
