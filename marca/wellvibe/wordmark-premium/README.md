# Wellvibe Wordmark Premium

Direccion: solo letras, sin simbolo y sin subtitulo.

## Criterio

La marca debe sentirse:

- premium
- editorial
- clinica contemporanea
- sobria
- confiable
- no wellness generico

## Versiones

- `wellvibe-wordmark-01-serif.svg`: editorial premium, mas elegante.
- `wellvibe-wordmark-02-sans.svg`: clinica moderna, mas limpia. **Direccion elegida.**
- `wellvibe-wordmark-03-condensed.svg`: mas de canal/contenido, fuerte en miniaturas.
- `wellvibe-wordmark-board.svg`: comparativo rapido.

## Nota

Estas versiones usan fuentes del sistema como aproximacion visual. Para identidad final conviene elegir o licenciar una tipografia premium y convertir el logo a curvas.
