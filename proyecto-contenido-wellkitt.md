# Proyecto de contenido Wellkitt

Estado: documento base de estrategia. Para fechas, temas semanales y ejecucion usar la programacion de 90 dias y el timeline.

## Objetivo

Cambiar la estrategia principal de transmisiones largas en vivo por un sistema continuo de contenido corto, pregrabado y basado en problemas reales del publico, usando IA para acelerar guiones, respuestas, anuncios y materiales.

El objetivo no es solo publicar mas contenido, sino crear un circuito completo:

1. Detectar problemas reales de la audiencia.
2. Convertir esos problemas en videos cortos.
3. Publicar de forma constante.
4. Usar cada contenido para abrir conversaciones.
5. Dar seguimiento por WhatsApp.
6. Convertir interes en consultas, cursos, productos o materiales.

## Principios de la nueva estrategia

- Menos transmisiones largas, mas piezas cortas y especificas.
- Cada video debe resolver un problema concreto.
- Cada contenido debe tener una accion siguiente clara.
- YouTube funciona como biblioteca y autoridad.
- TikTok/Reels/Shorts funcionan como descubrimiento.
- WhatsApp funciona como seguimiento y conversion.
- Zoom o eventos presenciales funcionan como cierre, confianza y venta.
- La IA ayuda a producir, ordenar y responder, pero la estrategia humana decide.

## Formato base de contenido

Cada video debe durar entre 8 y 15 minutos.

Estructura recomendada:

1. Apertura directa: nombrar el problema.
2. Contexto breve: por que le pasa a la gente.
3. Explicacion simple: que debe entender la persona.
4. Solucion inicial: 2 o 3 pasos practicos.
5. Puente a Wellkitt: producto, consulta, curso, manual o Zoom.
6. Cierre: invitacion especifica a escribir por WhatsApp.

Ejemplo:

> Hoy vamos a hablar de diarrea frecuente: cuando no es solo algo que comiste, sino una senal de irritacion, estres digestivo o desequilibrio. Te voy a decir que revisar, que evitar y cuando conviene pedir orientacion.

## Calendario semanal sugerido

### Lunes

Video de problema frecuente.

Ejemplos:

- Diarrea constante.
- Cansancio aunque duermas.
- Inflamacion abdominal.
- Dolor de cabeza recurrente.

### Miercoles

Video de producto o solucion.

Ejemplos:

- Epacril: para que casos puede tener sentido.
- Maca peruana y energia.
- Coenzima Q10 y circulacion.
- Sueros: cuando si y cuando no.

### Jueves

Actividad de conexion.

Puede ser:

- Zoom de producto.
- Taller breve presencial.
- Sesion de preguntas.
- Presentacion de un curso proximo.

### Sabado

Video de curso, caso o material.

Ejemplos:

- Que aprenderas en el curso de mascotas.
- Como usar mejor tu manual.
- Caso educativo: de una pregunta a una estrategia.
- Invitacion a actividad presencial.

## Flujo semanal de produccion

### Dia 1: Analisis de mensajes

Revisar WhatsApp del dia anterior y clasificar cada conversacion.

Categorias:

- Consulta.
- Producto.
- Curso.
- Pregunta de salud.
- Queja o problema.
- Seguimiento pendiente.
- Persona nueva.
- Persona recurrente.

Salida del analisis:

- Numero total de mensajes revisados.
- Temas mas repetidos.
- Productos mas preguntados.
- Cursos mas preguntados.
- Personas que requieren seguimiento.
- Ideas de contenido detectadas.

### Dia 2: Seleccion de temas

Elegir 3 o 4 problemas reales para la semana.

Criterios:

- Se repitio en WhatsApp.
- Puede conectar con un producto o consulta.
- Puede convertirse en video corto.
- Puede abrir una conversacion util.
- Puede derivar a Zoom, curso o material.

### Dia 3: Produccion con IA

Para cada tema, generar:

- Guion de video.
- Titulo de YouTube.
- Descripcion.
- Texto para WhatsApp.
- Texto para Facebook.
- Texto para TikTok/Reels.
- Mini anuncio o llamada a la accion.
- Ideas de material descargable o vendible.

### Dia 4: Grabacion

Grabar 3 o 4 videos en bloque.

Recomendaciones:

- Usar el mismo set.
- Cambiar solo pequenos elementos visuales si hace falta.
- Grabar videos de 8 a 15 minutos.
- Dejar espacio para cortes.
- Grabar tambien 2 o 3 frases cortas para Shorts/Reels.

### Dia 5: Programacion y seguimiento

Programar publicaciones.

Despues de publicar:

- Revisar comentarios.
- Revisar mensajes de WhatsApp.
- Clasificar interesados.
- Mandar respuestas personalizadas.
- Registrar oportunidades.

## Prompts de IA

### Prompt para detectar temas desde WhatsApp

```text
Actua como estratega de contenido y ventas para una clinica/escuela de bienestar llamada Wellkitt.

Voy a darte una lista de conversaciones resumidas de WhatsApp. Necesito que las clasifiques y detectes oportunidades.

Para cada conversacion identifica:
- Tipo: consulta, producto, curso, queja, seguimiento o pregunta general.
- Problema principal de la persona.
- Producto, curso o servicio relacionado.
- Respuesta sugerida en tono humano, breve y profesional.
- Si conviene seguimiento y en cuantos dias.
- Si de esta conversacion puede salir una idea de video.

Despues dame:
- Los 5 temas mas repetidos.
- 3 ideas de videos cortos.
- 3 oportunidades de venta sin sonar agresivos.
- 1 propuesta de Zoom o actividad presencial para esta semana.

Conversaciones:
[PEGAR AQUI]
```

### Prompt para crear guion de video corto

```text
Actua como guionista de videos educativos de salud natural para Wellkitt.

Tema:
[TEMA]

Publico:
Personas adultas interesadas en bienestar, salud natural, cursos y productos, con distintos niveles de conocimiento.

Objetivo:
Crear un video de 8 a 15 minutos que explique un problema real de forma clara, humana y practica, sin sonar como clase pesada ni como venta agresiva.

Estructura:
1. Apertura directa con el problema.
2. Explicacion sencilla.
3. Errores comunes.
4. 2 o 3 recomendaciones iniciales.
5. Puente natural hacia Wellkitt: consulta, producto, curso, Zoom o manual.
6. Cierre con llamada a escribir por WhatsApp.

Dame:
- Titulo principal.
- 5 titulos alternativos.
- Guion completo.
- Version corta de 60 segundos.
- Texto para descripcion de YouTube.
- Texto para WhatsApp.
- Texto para Facebook.
- 3 frases para convertir en Shorts/Reels.
```

### Prompt para respuesta personalizada de venta educativa

```text
Actua como asistente de seguimiento de Wellkitt.

Una persona pregunto esto:
[MENSAJE]

Producto o servicio relacionado:
[PRODUCTO/SERVICIO]

Dame:
- Respuesta corta para WhatsApp.
- Pregunta de seguimiento.
- Mensaje de seguimiento para 2 dias despues.
- Si aplica, una recomendacion de contenido o Zoom relacionado.
```

## Primer ciclo piloto

Duracion: 2 semanas.

Objetivo: probar el sistema antes de escalar.

### Semana 1

- Analizar WhatsApp de 3 dias anteriores.
- Detectar 5 temas frecuentes.
- Elegir 3 videos.
- Crear guiones con IA.
- Grabar 3 videos.
- Programar publicaciones.
- Hacer un reporte de mensajes generados por esos videos.

### Semana 2

- Repetir el analisis de WhatsApp.
- Ajustar temas segun lo que si genero respuesta.
- Hacer un Zoom o actividad presencial sobre un producto.
- Dar cupon o beneficio presencial.
- Medir resultados.

## Indicadores a medir

- Videos publicados por semana.
- Mensajes nuevos recibidos por contenido.
- Preguntas por producto.
- Preguntas por curso.
- Personas interesadas en consulta.
- Ventas relacionadas con el contenido.
- Asistencia a Zoom o actividad presencial.
- Personas que requieren seguimiento.
- Temas que mas interesan.

## Roles propuestos

### Direccion estrategica

- Define enfoque de salud, productos y cursos.
- Valida guiones.
- Graba videos.
- Dirige Zoom o actividades.

### Analisis y seguimiento

- Revisa WhatsApp diario.
- Clasifica conversaciones.
- Hace reportes.
- Prepara mensajes con IA.
- Detecta temas para contenido.
- Da seguimiento a interesados.

### Produccion

- Genera guiones con IA.
- Prepara anuncios.
- Programa videos.
- Corta clips cortos.
- Publica en redes.

## Decision importante

La transmision larga en vivo ya no debe ser el centro.

El centro debe ser un sistema semanal:

```text
WhatsApp -> Problemas reales -> Videos cortos -> Publicaciones -> Conversaciones -> Seguimiento -> Consulta/curso/producto/material
```

Ese sistema debe repetirse cada semana hasta que Wellkitt tenga una biblioteca viva de problemas, soluciones, productos y cursos.
