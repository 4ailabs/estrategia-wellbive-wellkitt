# Fondos virtuales Wellbive

Fondos generados para usar con chroma / pantalla verde en videos Wellbive.

## Fondos guardados

### fondo-01-consultorio-wellbive-dia.png

Uso recomendado:

- Videos largos educativos.
- Teleprompter.
- Temas generales de salud, sistema nervioso, digestion, energia, hormonas y defensas.

Estado:

- Aprobado como primer fondo base.
- Ajuste solicitado para siguientes versiones: evitar que se vea luz directa del sol o ventana protagonista.

Direccion visual:

- Consultorio profesional.
- Iluminacion suave de estudio.
- Clinico calido.
- No debe parecer casa.

### fondo-02-consultorio-wellbive-dia-sin-sol.png

Uso recomendado:

- Videos largos educativos.
- Grabaciones con teleprompter.
- Version mas controlada del consultorio, sin luz directa del sol.

Estado:

- Guardado como version mejorada.
- Ajuste pendiente para nuevas versiones: evitar mesa frontal o escritorio que ocupe primer plano.

Direccion visual:

- Iluminacion de estudio suave.
- Sin ventana protagonista.
- Sin rayos solares visibles.
- Consultorio profesional sobrio.

## Regla para siguientes fondos

Usar iluminacion suave y pareja, sin rayos solares visibles, sin ventana brillante y sin sombras dramaticas.
Evitar mesas frontales grandes; el fondo debe quedar detras del presentador, no competir con el cuerpo.
