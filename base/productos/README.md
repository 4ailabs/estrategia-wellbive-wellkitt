# Productos Wellkitt

Esta carpeta contiene la base de consulta de productos para mejorar guiones, temas semanales, Jueves Wellkitt, WhatsApp y materiales educativos.

## Como usar esta base

Primero consultar el indice editorial:

- `indice-editorial-productos.md`

Despues, si se necesita mas detalle, revisar los archivos fuente:

- `catalogo/base-conocimiento-productos.json`
- `catalogo/wellkitt-knowledge-base-completa.json`
- `catalogo/LISTA_COMPLETA_PRODUCTOS_WELLKITT.md`
- `catalogo/PRODUCT_URLS.md`

## Regla editorial

Los productos no deben presentarse como cura, diagnostico o sustituto de atencion medica.

En contenidos publicos usar este enfoque:

```text
Problema real -> patron de la persona -> orientacion -> ruta de apoyo -> producto o kit si aplica
```

Evitar este enfoque:

```text
Producto -> promesa -> compra inmediata
```

## Actualizacion

Cuando cambie el inventario, precio, URL o descripcion de productos en la tienda Wellkitt, actualizar primero la fuente original y despues traer una copia nueva a esta carpeta.
