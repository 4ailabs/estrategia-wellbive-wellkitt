## Uso recomendado

Este indice sirve para conectar productos con temas de contenido, semanas del calendario, palabras clave y rutas de seguimiento.

No es una guia medica. Es una herramienta editorial y comercial para ordenar mensajes.

## Reglas de comunicacion

- Hablar de apoyo, ruta, orientacion y seguimiento.
- Evitar promesas como "cura", "elimina", "garantiza" o "trata".
- Si hay medicamentos, embarazo, enfermedad cronica, sintomas intensos o tratamiento activo, recomendar orientacion personalizada.
- Presentar kits como rutas de 30, 60 o 90 dias, no como compra impulsiva.
- Cada video debe partir del problema de la persona, no del producto.

## Preguntas para perfeccionar un guion con productos

Antes de mencionar un producto, responder:

1. Que problema real esta expresando la persona.
2. Que patron queremos ayudarle a observar.
3. Que producto o kit podria ser puente.
4. Que advertencia o cuidado debe decirse.
5. Que siguiente accion queremos: WhatsApp, clinica, Zoom, consulta, curso o tienda.

## Formula para integrar producto sin que el canal parezca tienda

```text
80% educacion del problema
10% orientacion sobre patrones
10% ruta Wellkitt: clase, tienda, producto, kit, consulta o curso
```

Ejemplo:

```text
No vamos a elegir producto por moda. Primero identificamos el patron: ansiedad, insomnio, cansancio, tension o ruido mental. Despues vemos si tiene sentido una ruta de apoyo como CALMA.
```
