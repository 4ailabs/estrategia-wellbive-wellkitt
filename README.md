## Estrategia central

La estrategia no es vender productos de forma aislada. El sistema debe partir de problemas reales de las personas:

```text
Problema real -> video de YouTube -> Shorts -> Jueves Wellkitt en Clinica + Zoom -> tienda fisica + cupon -> ficha -> WhatsApp -> producto / kit / consulta / curso
```

## Ritmo semanal

- Lunes: video educativo basado en un problema frecuente.
- Martes: Short de atraccion.
- Miercoles: video puente hacia producto, curso o actividad.
- Jueves: Jueves Wellkitt en Clinica + Zoom.
- Viernes: seguimiento por WhatsApp, cupon y cierre.
- Sabado: Short, testimonio o recordatorio.

## Reglas para colaboradores

- No cambiar la programacion de 90 dias sin actualizar tambien el timeline y el tablero.
- Cada semana debe tener una palabra clave clara para WhatsApp.
- Cada producto debe presentarse como parte de una ruta o solucion, no como frasco aislado.
- Los vivos largos no son el centro de la estrategia; pueden usarse solo como apoyo comunitario.
- La clinica es el centro de experiencia; Zoom es extension para personas lejanas.
