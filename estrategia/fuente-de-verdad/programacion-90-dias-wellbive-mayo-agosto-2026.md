# Programacion 90 dias Wellbive / Wellkitt / Instituto

Periodo: lunes 25 mayo a domingo 23 agosto 2026

## Decision estrategica

Tres meses es el periodo correcto para experimentar.

Menos de 8 semanas no alcanza para ver patrones reales. Mas de 3 meses sin evaluar puede hacer que se sostengan formatos que no venden. La recomendacion es:

```text
90 dias de prueba seria
-> revision de metricas
-> duplicar lo que funcione
-> eliminar lo que no genere flujo, mensajes o ventas
```

Este periodo debe medir tres cosas:

1. Si YouTube genera mensajes.
2. Si los Jueves Wellkitt generan visitas a clinica.
3. Si las visitas se convierten en venta, curso, consulta o seguimiento.

## Estructura semanal fija

| Dia | Accion | Objetivo |
|---|---|---|
| Lunes | Video problema en YouTube | Atraer desde una necesidad real |
| Martes | Short del problema | Descubrimiento |
| Miercoles | Video herramienta/producto/kit | Educar y preparar venta |
| Jueves | Jueves Wellkitt en Clinica + Zoom | Crear flujo presencial |
| Viernes | Seguimiento WhatsApp + cupón | Convertir |
| Sabado | Short/resumen/caso | Mantener conversacion |
| Domingo | Encuesta + analisis de datos | Definir siguiente tema |

## Reglas del experimento

- Cada semana debe tener una palabra clave.
- Cada Jueves Wellkitt debe tener producto protagonista, kit o mesa fisica.
- Cada asistente presencial debe llenar ficha.
- Cada cupón presencial debe tener nombre, vigencia y categoria.
- Cada tema debe conectar con un producto y, si aplica, con curso o consulta.
- El canal no debe titularse desde productos; se titula desde problemas.
- El reporte semanal decide el siguiente ajuste.

## Fases

### Fase 1: Instalacion del sistema

Semanas 1 a 4.  
Objetivo: cambiar el habito de vivo largo a sistema semanal.

### Fase 2: Validacion comercial

Semanas 5 a 9.  
Objetivo: medir que temas llevan gente a clinica y generan compras.

### Fase 3: Escalamiento y cursos

Semanas 10 a 13.  
Objetivo: duplicar formatos ganadores y conectar con cursos de julio-agosto.

## Semana 14 opcional: 24-30 agosto

Tema madre: evaluacion y repeticion ganadora.

Esta semana no debe inventar un tema nuevo. Debe repetir el tema que mas genero:

- Visitas presenciales.
- Mensajes de WhatsApp.
- Ventas.
- Interesados en curso.
- Solicitudes de consulta.

La repeticion puede ser:

```text
Los 5 casos mas frecuentes que vimos en estos 90 dias
```

O:

```text
La ruta que mas pidieron: [DIGESTION / ESTRES / ENERGIA / PESO]
```

## Decision despues de los 90 dias

Si funciona:

- Mantener Jueves Wellkitt.
- Repetir los 4 modulos ganadores.
- Convertir los temas fuertes en cursos de un dia.
- Crear paginas web por modulo.
- Crear kits oficiales por problema.

Si no funciona:

- No abandonar el sistema completo.
- Revisar si fallo el tema, la convocatoria, la oferta, el seguimiento o la experiencia presencial.

La pregunta no debe ser "funciono o no funciono".

La pregunta correcta es:

```text
Que parte del embudo se rompio?
```
