# Guia para colaboradores

## Antes de editar

Revisar primero:

- `README.md`
- `programacion-90-dias-wellbive-mayo-agosto-2026.md`
- `timeline-wellbive-90dias.html`
- `tablero-estrategia-wellbive.html`

## Cambios de calendario

Si se cambia una semana, actualizar los documentos relacionados:

- Programacion madre: `programacion-90-dias-wellbive-mayo-agosto-2026.md`
- Calendario operativo: `calendario-operativo-wellbive-junio-julio-2026.md`
- Timeline visual: `timeline-wellbive-90dias.html`
- Tablero estrategico: `tablero-estrategia-wellbive.html`

## Criterio editorial

Cada contenido debe responder:

1. Que problema real atiende.
2. Que video o Short lo introduce.
3. Que palabra clave activa WhatsApp.
4. Que producto, kit, consulta, curso o actividad sigue.
5. Como se medira el resultado.

## Formato de propuesta

Para proponer un nuevo tema:

```text
Problema:

Jueves Wellkitt:
Producto / kit:
Curso o actividad relacionada:

Meta:
```
