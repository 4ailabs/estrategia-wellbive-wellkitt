# Wellbive Inside

*Ciencia clinica para tu cuerpo*

Publicacion quincenal del Instituto Centrobioenergetica.

---

## Concepto

Revista digital quincenal que integra el contenido de las semanas de produccion (videos, Jueves en Wellkitt, investigacion, perfiles) con articulos originales, noticias de ciencia y casos clinicos.

## Frecuencia

Quincenal. Cada numero cubre 2 semanas de contenido.

## Formato

PDF digital, 8-12 paginas. Distribucion por WhatsApp, email y link en descripcion de YouTube.

---

## Estructura fija por numero

| Seccion | Contenido | Fuente |
|---|---|---|
| Portada | Tema principal + titulo | Tema de la quincena |
| Editorial | Reflexion breve del Dr. Ojeda (1 pagina) | Nuevo, voz personal |
| Articulo de ciencia | El hallazgo mas fuerte de la quincena, explicado a fondo | investigacion-*.md de las semanas |
| Los perfiles | "Cual es tu tipo" — framework de la quincena con autotest | sistema-perfiles-wellbive.md |
| Planta de la quincena | Una planta medicinal a fondo: origen, molecula activa, mecanismo, para que perfil aplica, como se usa en el Instituto | Catalogo Wellkitt + investigacion |
| Caso clinico | Caso real anonimizado: como llego, que se observo, que ruta se siguio, que cambio | Practica clinica del Instituto |
| Que probamos en Wellkitt | Resumen del Jueves con fotos y testimonios | material-jueves/ + fotos reales |
| Producto a fondo | 1 producto explicado con mecanismo y para quien es | Catalogo Wellkitt |
| Noticias breves | 3-4 notas cortas de salud/ciencia/regulacion | Investigacion web |
| Agenda | Proximos Jueves en Wellkitt + palabra clave + como llegar | Calendario de semanas |
