# Teleprompter semana 01

Scripts palabra por palabra para grabacion.

## Velocidades sugeridas

- Video principal: 125-135 palabras por minuto.
- Shorts: 130-150 palabras por minuto.

## Archivos

- `video-principal-tu-cuerpo-sigue-en-emergencia-12min.md`
- `short-01-ruido-mental-45s.md`
- `short-02-duermo-pero-no-descanso-40s.md`
- `short-03-cuerpo-en-alerta-50s.md`

## Regla de produccion

Todo video debe tener dos archivos:

1. Guion estrategico: estructura, objetivo, CTA y descripcion.
2. Teleprompter: texto hablado palabra por palabra segun duracion asignada.
