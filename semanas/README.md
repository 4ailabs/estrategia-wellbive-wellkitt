# Semanas de produccion

Estas carpetas sirven para ejecutar la estrategia semana por semana.

La programacion madre sigue siendo:

- `../programacion-90-dias-wellbive-mayo-agosto-2026.md`
- `../timeline-wellbive-90dias.html`
- `../tablero-estrategia-wellbive.html`

## Como usar cada semana

En cada carpeta se debe desarrollar:

- Guion del video principal.
- Shorts de la semana.
- Texto para WhatsApp.
- Material del Jueves Wellkitt.
- Lista de productos o kit protagonista.
- CTA y palabra clave.
- Reporte de resultados.

## Estructura recomendada dentro de cada semana

```text
README.md
guiones/
shorts/
whatsapp/
material-jueves/
reporte/
```

No todas las subcarpetas tienen que existir desde el inicio. Se crean cuando haya contenido real.
